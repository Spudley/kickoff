Rules

There is already a small set of rules, which you can use.

Security Rules

HttpHeaderXSSProtectionPresent

Tests, whether the X-XSS-Protection HTTP header is present. More Infos about X-XSS-Protection header

HttpHeaderXSSProtectionSecure

Tests, whether the X-XSS-Protection HTTP header is present and has value "1; mode=block". More Infos about X-XSS-Protection header

HttpHeaderExposePHP

Tests, whether the X-Powered-By HTTP header is present. More Infos about expose_php in php.ini.

HttpHeaderContentTypeNoSniffing

Tests, whether the X-Content-Type-Options HTTP header is present and has value "nosniff". More Infos about X-Content-Type-Options header

HttpHeaderFrameOptionsSameOrigin

Tests, whether the X-Frame-Options HTTP header is present and has value "SAMEORIGIN". More Infos about X-Frame-Options header

HttpHeaderCookieWithHttpOnlyFlag

Tests, whether the Session cookie has the flag "httpOnly" set. More infos about httpOnly cookie flag.

HttpHeaderCookieWithHttpSecureFlag

Tests, whether the Session cookie has the flag "Secure" set. More infos about secure cookie flag.

HttpHeaderHSTSPresent

Tests, whether the "Strict-Transport-Security" HTTP header is present. More Infos about HSTS.

HttpHeaderHSTSWithSubdomains

Tests, whether the "Strict-Transport-Security" HTTP header is present and has the "includeSubDomains" flag set. More Infos about HSTS.

Seo and performance Rules

HttpRequestTime

Tests, whether the website was downloaded in less than a sepcified amount of seconds.

HttpHeaderHasFarFutureExpiresHeader

Tests, whether the website or resource sends a far future "Expires" header.

HttpHeaderResourceFound

Tests, whether the website or resource sends a 200 status code.

HttpHeaderResourceIsMissing

Tests, whether the website or resource, which is missing sends a 404 status code.

HttpHeaderResourceIsGzipped

Tests, whether the website or resource sends gzip encoded content.

TitleTagLength

Tests, whether the title tag length is in specified boundaries.

MetaDescriptionLength

Tests, whether the meta description length is in specified boundaries.
